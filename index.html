概述

iView 是一个基于VUE的前端UI开源组件库，具备良好的用户体验。

为了方便在项目的初期规划过程中，能够更直观的体现出项目结果，提升各个业务系统的体验设计统一性，

特整理归纳相关Axure组件模板、备注用法、设计原则及建议，方便在制作原型时使用，提升最终业务系统用户体验。（本文档在线实时更新，请收藏链接）

关于 iView

iView 是一套基于 Vue.js 的开源 UI 组件库，主要服务于 PC 界面的中后台产品。


特性 ：

• 高质量、功能丰富

• 友好的 API ，自由灵活地使用空间

• 细致、漂亮的 UI

• 事无巨细的文档

• 可自定义主题

• 同时具有PC端+H5移动端相关组件

点此访问iView项目地址

关于设计与文案

为了保证原型的实现效果，组件库中使用的所有组件大都是按照iView项目的设计进行实现；

库中的文案很多也是沿用了官方文档中的说明，以保证观点的延续；

补充的使用指南、设计原则、设计建议、常用模板等，可适用于本公司所有后后台系统！

预览之前

由于组件库使用了特定的字体和图标，如果您的电脑中没有安装对于的字体，可能会出现大量的显示错乱和问号；

建议开始之前先下载字体文件
